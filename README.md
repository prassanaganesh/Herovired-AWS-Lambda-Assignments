# Herovired AWS Lambda Assignments

This repository contains solutions for Herovired AWS Lambda Assignments using **AWS Lambda**, **Boto3 (Python SDK)**, and supporting AWS services (EC2, S3, IAM,Lambda& EventBridge).

---

└── README.md   (this file)


---

### 📘 Assignments Included

- [Assignment 1: EC2 Auto Management](Assignment1-EC2-AutoManagement/README.md)  
  *Automatically stop/start EC2 instances based on tags.*

- [Assignment 2: S3 Cleanup](Assignment2-S3-Cleanup/README.md)  
  *Delete files older than 30 days from an S3 bucket.*

- [Assignment 3: S3 Encryption Monitor](Assignment3-S3-Encryption-Monitor/README.md)  
  *Detect unencrypted S3 buckets and log them for review.*

- [Assignment 5: EC2 Auto Tagging](Assignment5-EC2-AutoTagging/README.md)  
  *Automatically tag new EC2 instances with launch date and custom tags.*

---
### Technologies Used
  - AWS Lambda (Python 3.12 runtime)
  - Boto3 SDK for AWS service interactions
  - IAM Roles & Policies for permissions
  - Amazon EC2 for instance management
  - Amazon S3 for storage and cleanup tasks
  - Amazon EventBridge for event-driven automation
---
### Results
  -  Each assignment demonstrates automation of AWS tasks:
  -  EC2 lifecycle management via tags.
  -  Automated cleanup of S3 buckets.
  -  Monitoring encryption compliance for S3.
  -  Auto-tagging EC2 instances on launch.
